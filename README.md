# POTATO throwaaa
You throw potatoes at NPCs, and yeah get as many kills as you want !!

## Controls

- WASD to move around
- Spacebar to jump 
- C to clear out potatoes if it lags (they dispawn too after a while)
- TAB to open shop ! 
- B to toggle big potato (if bought)
- M to toggle machinegunnnnn (if bought)

have fun !
